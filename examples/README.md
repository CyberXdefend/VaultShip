# VaultShip Examples

This folder contains practical examples for local development and DevOps workflows.

## Contents

- `dev/docker-compose.yml` - minimal service compose file
- `dev/vaultship.toml` - local developer config
- `dev/commands.sh` - end-to-end local command sequence
- `devops/vaultship.toml` - production-oriented config baseline
- `devops/license-issuance.sh` - operator license flow
- `devops/pipeline.sh` - CI-friendly build/push/pull/inspect flow
- `sdk/main.rs` - minimal runtime check integration

## Quick Start

```bash
bash examples/dev/commands.sh
```
