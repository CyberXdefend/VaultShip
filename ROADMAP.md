# VaultShip Roadmap

## Phase 1 (Weeks 1-6): Foundation
- Hardening engine (seccomp, capabilities, read-only rootfs)
- Binding engine (hardware fingerprint + machine authorization checks)
- Encryption engine (AES-256-GCM layer primitives)
- Unified CLI integration for VaultShip-protected containers

## Phase 2 (2026 H2): Enterprise Runtime Controls
- challenge-response offline activation bundles
- runtime policy profiles for deployment environments
- stronger provenance verification and controls
- TPM-backed key storage experiments

## Phase 3 (2027): Confidential Workloads
- confidential computing feasibility and prototypes
- attestation-gated key release flows
- hardened secret bootstrap patterns

## Phase 4 (Future): Fleet Platform
- Kubernetes operator support
- web-based policy and binding management
- fleet-wide policy orchestration and telemetry

## Phase 5 (Future): Ecosystem Expansion
- multi-language SDKs
- enterprise integrations and SIEM connectors
- managed update channels and policy packs
